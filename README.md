# 🛒 E-Commerce Frontend Project

This is a final portfolio project for the Frontend Development course. It is a fully responsive e-commerce web application built with React and Tailwind CSS.

## 🚀 Features
- **Product Catalog**: Fetches real products from Fake Store API.
- **Dynamic Routing**: Navigates between Home, Cart, and Product Details using React Router.
- **State Management**: Uses React Context API and LocalStorage to persist cart data.
- **Checkout Validation**: Custom form validation for user details.
- **Responsive Design**: Mobile-first UI using Tailwind CSS.

## 🛠 Tech Stack
- React 18 (Vite)
- Tailwind CSS
- React Router DOM
- Axios

## ⚙️ Setup Instructions
1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
